# Analysis Report : Amazon Sales Data

## Executive Summary
- Total Revenue : <b> 7.85 CR INR </b>
- Best Performing Category : <b>Set</b>
- Top State (in Revenue) : <b>Maharashtra</b>
- Cancelled Orders (in 4 months): <b>14.21%</b>

## Key Insights and Recommendations
1. Orders count for Categories Bottom and Saree are decreasing continously. Since these are not the wedding season, Bottoms should be considered first for solutions.

2. Number of Cancelled Orders increased drastically between month 3 and 4 but started decreasing thereafter. The trend should be monitored and corrective actions should be taken in case there is a rise again.

3. Revenue from the Southern States is strong. We should try maximising the revenue even more in this region.
